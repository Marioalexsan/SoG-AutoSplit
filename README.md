# SoG-AutoSplit

LiveSplit Autosplitter for Secrets of Grindea

## Contributors

[3ps1l0n](https://www.speedrun.com/users/3ps1l0n)

## Features

* Load Removal
* Cutscene Removal
* Autosplitting for Story Mode and Arcade Mode
* Automatic Start
* Automatic Resets
